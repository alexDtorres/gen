# Getting Started

`go get -u github.com/alexDtorres/gen`


## Example

```
package main

import "github.com/alexDtorres/gen"

func main() {
  pkg := gen.Install()
  pkg.Apply()
}
```


## Installation

`go install github.com/alexDtorres/gen/cli/app@latest`


## CLI Usage

`app gen <your_app_name>`
